为什么 AI 开发者需要星栈？

每天生成大量 HTML 页面，传统部署流程太慢。星栈把发布简化到一次拖拽或一个 API 调用。

🤖

AI 生成的页面

Claude Code、Codex、OpenClaw 产出的 HTML 直接上传，秒得全球可访问 URL。

🚀

一键发布

不用买服务器、配域名、设 CDN。拖拽文件或粘贴 HTML，自动分发到 5 节点集群。

🌐

全球就近访问

新加坡、东京、首尔、曼谷、美国 5 节点分布，亚洲用户优先命中亚洲节点。

♾️

内容寻址

基于 IPFS CID，文件不可篡改。更新站点即更新 CID，天然支持版本回滚。

🛡️

抗审查与高可用

单节点故障不影响访问，数据由多个节点共同保存，无需担心服务器被墙。

Claude Code / Codex 集成示例
# 一键发布 HTML 文件到星栈
curl -X POST "https://starstack.store/api/ipfs-upload" \ -F "file=@index.html" | python3 -m json.tool

# 返回示例：{"gateway":"https://starstack.store/ipfs/Qm..."}
# 然后用浏览器访问 https://starstack.store/ipfs/Qm...
// JavaScript: 在 AI 生成的页面里直接上传
const form = new FormData();
form.append('file', new Blob([htmlString], {type: 'text/html'}), 'index.html');
const res = await fetch('https://starstack.store/api/ipfs-upload', {
method: 'POST', body: form
});
const { gateway } = await res.json();
console.log('Published:', gateway);
